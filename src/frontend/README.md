# Smart Shopping Planner - Frontend

A mobile-first grocery shopping schedule planner with nearby store discovery, smart scheduling, budget tracking, and bilingual UI (English/Hindi).

## Features

- **User Input**: Location (manual or geolocation), date/time, budget, and items (with voice input support)
- **Shop Discovery**: Find nearby grocery stores using OpenStreetMap (free) or Google Places API (optional)
- **Smart Recommendations**: Top 3 shops ranked by availability, distance, and ratings
- **Cost Estimation**: Budget tracking with India-focused price estimates
- **Schedule Export**: Download as calendar event (.ics) or PDF
- **Past Schedules**: Save, view, and clone previous shopping trips
- **Bilingual UI**: Toggle between English and Hindi
- **Demo Mode**: Test with sample Indore, India data without API keys
